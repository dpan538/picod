# Picod Plant Layer Direction

Version: 0.1  
Purpose: align plant asset language and placement so the map reads as Japanese garden + ceremonial ecology, not generic icon scatter.

## Core Principle
Plant readability is driven by silhouette first, color second, micro-detail last.

## Plant Roles
- Sacred Evergreen: vertical, stable framing around shrine/torii/pagoda approach lines.
- Seasonal Landmark: high-recognition accents (sakura, hydrangea-like flower clusters).
- Garden Structure: shrubs and low greens that organize edges/yards.
- Forest Edge: irregular, mixed green mass near map perimeter.

## Placement Rules
- Avoid even spread. Prefer clusters, bands, and framed edges.
- Ceremonial zones: restrained center, richer edge framing.
- Residential zones: lived-in shrubs + clustered flowers near paths/yards.
- Waterside zones: reeds + low dense flower clusters.
- Forest perimeter: denser, irregular green with mixed heights.

## Sakura Rule
- No perfect circular canopy.
- Must include visible trunk logic.
- Blossom mass should read as irregular cloud or light droop.
- Prioritize silhouette over petal noise.

## Sacred Evergreen Rule
- Vertical profile should be stronger than decorative trees.
- Keep canopy compact and non-round to avoid toy-like reads.
- Use fewer placements with clearer intent (approach framing, precinct edges).

## Flower Cluster Rule
- Prefer grouped patches over evenly spaced single flowers.
- Flower beds can anchor cluster centers.
- Cluster color should support place identity, not confetti noise.
